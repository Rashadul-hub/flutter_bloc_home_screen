A small Flutter demo app that shows how to structure a Home screen using the BLoC (flutter_bloc) pattern.

## Overview

This repository demonstrates a clean and simple approach to managing state on a Home screen using the BLoC pattern and the flutter_bloc package. It's intended as a starting point for learning how to separate presentation, business logic, and state in Flutter apps.

## Features

- Flutter (stable)
- State management using flutter_bloc
- Simple, testable Home screen architecture
- Responsive layout and theming-ready
